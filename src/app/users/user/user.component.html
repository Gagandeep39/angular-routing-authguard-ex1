Above button will change URL path but wont update data This is the dfault behaviour of Angar If a route is called frm same route, the contents wont be rerendered

To fix this issue, we use 'route.params' which is a part of Observable. We subscribe to params object

We can still use activatedParams without subscription if we are sure that there is only one way to reach the path and that is from another path
